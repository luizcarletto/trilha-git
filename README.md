# trilha-git

PRINCIPAIS COMANDOS GIT:

- GIT ADD
   Ex: <nome do arquivo> / .* (extesão) / . (todos os arquivos)

- GIT COMMIT 
   Ex: -m "Mensagem"
 
- GIT DIFF
   Ex: --staged (mostra o que foi alterado e esta na staging area apos o add)
 
- GIT LOG
   Ex: -p (mostra o diff de cada arquivo) / -1 (mostra um unico commit) / -2 (mostra os dois ultimos commits) / -n (informar o numero de commits)
 
- GITK
 
- GIT TAG
   Ex: -a (tag anotada) / -m (mensagem) / -d (deletar tag)
 
- GIT SHOW
 
- GIT CHECKOUT
   Ex: --<nome do arquivo> (volta o arquivo para o status original do commit)
 
- GIT BRANCH <nome>
   Ex: -d (deletar branch) / git branch (lista todas as branchs)
 
- GIT MERGE
 
- GIT COMMIT
   Ex: --amend (edita o ultimo commit realizado) 

- GIT RESET HEAD
 
- GIT CLONE FILE

- GIT PUSH
 
- GIT PULL

- GIT FETCH
 
- GIT REMOTE
